# Products_Management
A Spring Boot RESTful API application for managing products. This project provides full CRUD operations for products with database integration.

## Features

- Create, Read, Update, and Delete products
- RESTful API endpoints
- MySQL/PostgreSQL database integration
- Swagger/OpenAPI documentation
- JWT-based authentication (if implemented)
- Docker support for easy deployment

## Technology Stack

- **Backend:** Java 17, Spring Boot, Spring Data JPA, Hibernate
- **Database:** MySQL / PostgreSQL
- **Security:** Spring Security, JWT & Refresh Tokens
- **Testing:** JUnit 5, Mockito
- **Documentation:** Swagger/OpenAPI
- **DevOps:** Docker, Docker Compose

## Prerequisites

- Java 17+
- Maven
- MySQL or PostgreSQL
- Git
- Docker (optional)
